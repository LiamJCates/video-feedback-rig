# Video Feedback Rig Knowledgebase

This repository serves as a **knowledgebase** and **lab notebook** for a video feedback synthesis rig. It systematically documents all aspects of the hardware devices, signal topology, creative experiments, behavioral quirks, and future gear targets.

## Purpose
The goal of this repository is to provide an organized structure that allows for easy updates and insights into the workings of the video feedback synthesis setup. It is optimized for:
- **Conversational updates with AI** to facilitate quick and efficient logging of changes and thoughts.
- **Mobile editing** to ensure that modifications can be made anytime, anywhere, boosting accessibility and ease of use.
- **Incremental documentation** that supports a continuous evolution of knowledge over time, reflecting the rig's growth and development.
